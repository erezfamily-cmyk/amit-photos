## 🤖 אוטומציות (GitHub Actions)

| Workflow | תדירות | מה עושה |
|---|---|---|
| **עדכון תמונות** | כל יום 09:00 | מושך תמונות חדשות מ-Google Drive → מעדכן photos.json → deploy Cloudflare |
| **Instagram** | ראשון/רביעי/שישי 12:00 | בוחר תמונה → Claude כותב כיתוב + hashtags → מפרסם |
| **Facebook** | שלישי/שישי 11:00 | בוחר תמונה → Claude כותב פוסט → מפרסם |
| **Pinterest** | כל יום 10:00 | מפרסם 3 תמונות ל-3 לוחות (לפי קטגוריה) |
| **מיגרציה ל-R2** | ידני | מעביר תמונות מ-Google Drive → Cloudflare R2 |

**הגיון Anti-repeat:** כל פלטפורמה שומרת קובץ JSON עם IDs שפורסמו. תמונה לא תפורסם שוב עד שתמה הרוטציה.

**Claude Vision:** Instagram + Facebook משתמשים ב-Claude Vision לניתוח ויזואלי של התמונה ולכתיבת כיתוב מותאם.

## 💡 המלצות לשיפור

### עדיפות גבוהה

1. **השלמת מיגרציה ל-R2**
   כרגע רק 12 מתוך 471 תמונות ב-D1/R2. המיגרציה פועלת אבל לאט. כל עוד תמונות מגיעות מ-Google Drive, הן פגיעות (URLs עם תפוגה, חסמי גישה). יש להשלים מיגרציה מלאה.

2. **טוקן פייסבוק מתחדש אוטומטית**
   הטוקן פג תוקף מדי פעם ומצריך פעולה ידנית. פתרון: שמירת Long-Lived Token + workflow שמחדש אותו לפני תפוגה.

3. **ניוזלטר — היסטוריית שליחות**
   אין כרגע היסטוריה של מיילים ששלחנו. קשה לדעת מה נשלח ומתי.

### עדיפות בינונית

4. **Pinterest — כיתובים עם Claude**
   Instagram ו-Facebook משתמשים ב-Claude Vision לכיתובים. Pinterest מפרסם עם תיאור בסיסי בלבד. שווה להוסיף כיתוב חכם ל-Pinterest גם.

5. **המלצות hardcoded**
   הציטוטים בסקשן "לקוחות ממליצים" כתובים ישירות ב-HTML. כשיהיו לקוחות אמיתיים — כדאי לנהל אותם מהאדמין.

6. **בחירה מרובה בגלריה**
   כרגע לא ניתן לבחור כמה תמונות יחד. שימושי לצפייה / הורדה / שיתוף מרובה.

7. **תצוגת סטטוס הזמנה ללקוח**
   לקוח שהזמין הדפסה מקבל רק מייל. אין עמוד "מצב ההזמנה שלי" שהוא יכול לבדוק.

### עדיפות נמוכה / עתיד

8. **Gelato Branded Inserts**
   פתק תודה ממותג בתוך האריזה — דורש Gelato+ מנוי + בטא. לשקול אחרי הזמנה ראשונה.

9. **Analytics לחנות**
   כמה אנשים פתחו lightbox → כמה לחצו "רכוש" → כמה השלימו. כרגע אין funnel tracking.

10. **Google Photos → Drive אוטומטי**
    כרגע עמית מעלה ידנית ל-Drive. אם יש אינטגרציה ישירה מ-Google Photos, זה יחסוך צעד.
