### גלריה

- פילטרים היררכיים: קטגוריות ראשיות + "מקומות בעולם" עם תת-קטגוריות (10 מדינות)
- חיפוש חופשי לפי שם תמונה
- Infinite scroll (100 תמונות ראשונות + טעינה בגלילה)
- **סדר תמונות:** תמונות עם `sort_order` מוצגות ראשון לפי סדר ידני; שאר התמונות בסדר אקראי שמתחלף כל שבוע
- **Lightbox** עם:
  - ניווט בחצים + מקלדת
  - תמונות קשורות מאותה קטגוריה
  - שיתוף WhatsApp / Facebook
  - כפתור "הוסף לסל" לחנות הדפסות
  - כפתור "ראה על הקיר" (mockup עם בחירת צבע קיר)
  - Progress bar, EXIF data
- הגנת תמונות: בלוק קליק ימני, גרירה, Ctrl+S, overlay שקוף
- SEO: alt text עברי אוטומטי לכל תמונה

### תמונת השבוע ⭐ (חדש)

**סטריפ מעל הגלריה** שמציג את תמונת השבוע הנבחרת:
- מוצג רק כשיש תמונת שבוע מוגדרת
- שורה עם תמונה ממוזערת + שם + "25% הנחה השבוע על כל גדלי הרכישה"
- כפתור **"הצג תמונה ▼"** — פותח/סוגר אזור מורחב עם התמונה בגדול
- באזור המורחב: שני כפתורים:
  - **"🖼 תצוגה על הקיר"** — פותח מודל wall-mockup להדמיית התמונה תלויה על קיר
  - **"רכוש עכשיו"** — פותח מודאל רכישה עם ההנחה מוחלת אוטומטית
- ההנחה (25% ברירת מחדל, ניתן לשינוי) חלה על מחירי ההורדה הדיגיטלית
- תמונת השבוע מסומנת ב-badge ⭐ בפינת הכרטיס בגלריה
- ממשק דו-לשוני: עברית + אנגלית

### חנות הדפסות (Print Shop)

**3 סוגי מוצרים דרך Gelato:**

| סוג | גדלים | מחיר |
|---|---|---|
| פוסטר (נייר מט 170gsm) | 20×25, 30×40, 40×50, A3, A2, 45×60, 60×90 | $34–$85 |
| קנבס (מסגרת עץ) | 20×20, 20×25, 30×40, 40×50, 45×60 | $38–$86 |
| מתכת (אלומיניום 3mm) | 30×30, 30×40, 30×45 | $70–$82 |

**תהליך הזמנה:**
1. בחירת מוצר + גודל
2. חיתוך תמונה (crop tool + בדיקת רזולוציה)
3. מילוי פרטי משלוח + תשלום PayPal
4. מייל אישור אוטומטי עם קישור ביטול
5. עדכון סטטוס ב-Webhook מ-Gelato

**תכונות:**
- תיבת חיתוך מסתובבת אוטומטית לפי orientation
- בדיקת רזולוציה (חוסמת הזמנה אם נמוכה מדי)
- הערת מכס כשמחיר ≥ $75
- Wall mockup — "ראה על הקיר"
- FAQ מפורט
- מייל שילוח אוטומטי ללקוח כש-shipped

## 🔧 ניהול (Admin) — admin.html

**כניסה:** סיסמה + session token (עם נעילה אחרי ניסיונות כושלים)

### לוח בקרה
- מד נרשמים / פניות / תמונות
- סטטוס מהיר

### תמונות
- רשימת כל התמונות ב-D1
- עריכת כותרת / קטגוריה / תיאור
- פרסום / הסרה מהגלריה (published toggle)
- מחיקה (כולל מחיקה מ-R2)
- העלאה ידנית (drag & drop) → R2 + D1
- יצירת כותרת עברית אוטומטית עם Claude
- **מחיר מיוחד לתמונה בודדת** — override על גדלי small/medium/large
- **סימון "חדש"** — מוסיף תג חדש על כרטיס בגלריה
- **בחירה מרובה (Bulk)** — סימון כמה תמונות → שינוי קטגוריה / מחיר / מחיקה בבת אחת
- **סדר ידני** — drag & drop לשינוי סדר הופעת תמונות בגלריה

### תמונת השבוע ⭐ (חדש)
- **הצע אוטומטית** — המערכת בוחרת תמונה אקראית מ-20% הפחות נמכרות
- **אשר** — מגדיר את התמונה כתמונת השבוע
- **הצע שוב** — אם ההצעה לא מוצאת חן
- **נקה** — מסיר את תמונת השבוע (הסטריפ נעלם מהאתר)
- ניתן לשינוי ה-discount (ברירת מחדל 25%)

### נרשמים לניוזלטר
- רשימה עם תאריך הרשמה ושם

### לקוחות ופניות
- כל הפניות מטופס צור קשר
- שינוי סטטוס מהיר (dropdown בשורה): ממתין / טופל / סגור
- כפתור "השב" → פותח Gmail עם מייל מוכן
- צפייה בפנייה מלאה (dialog)

### סטטיסטיקות
- גרף page views — 30 יום אחרונים
- פירוט לפי מדינה
- מעקב אוטומטי (Cloudflare Worker + D1)

### ניוזלטר
- עריכת נושא + גוף המייל
- תצוגה מקדימה (iframe)
- שליחה לכל הנרשמים

### רכישות (Purchases)
- הכנסה כוללת + פירוט לפי גודל (small/medium/large)
- גרף 30 יום
- 5 תמונות הכי נמכרות
- טבלת כל הטוקנים עם סטטוס

### הזמנות הדפסה
- טבלת כל הזמנות עם: תאריך, לקוח, טלפון, מייל, כתובת, מוצר, מחיר, Gelato ID, סטטוס
- כפתור 🔄 **רענן סטטוס** לכל הזמנה `in_production` — שולף מ-Gelato ומעדכן בזמן אמת
- כפתור "החזר ב-PayPal" להזמנות מבוטלות

## 🤖 אוטומציות (GitHub Actions)

| Workflow | תדירות | מה עושה |
|---|---|---|
| **עדכון תמונות** | כל יום 09:00 | מושך תמונות חדשות מ-Google Drive → מעדכן photos.json → deploy Cloudflare |
| **Instagram** | ראשון/רביעי/שישי 12:00 | בוחר תמונה → Claude כותב כיתוב + hashtags → מפרסם |
| **Facebook** | שלישי/שישי 11:00 | בוחר תמונה → Claude כותב פוסט → מפרסם |
| **Pinterest** | כל יום 10:00 | מפרסם 3 תמונות ל-3 לוחות (לפי קטגוריה) |
| **מיגרציה ל-R2** | ידני | מעביר תמונות מ-Google Drive → Cloudflare R2 |

**הגיון Anti-repeat:** כל פלטפורמה שומרת קובץ JSON עם IDs שפורסמו. תמונה לא תפורסם שוב עד שתמה הרוטציה.

**Claude Vision:** Instagram + Facebook משתמשים ב-Claude Vision לניתוח ויזואלי של התמונה ולכתיבת כיתוב מותאם.

## 💡 המלצות לשיפור

### עדיפות גבוהה

1. **השלמת מיגרציה ל-R2**
   תמונות שמגיעות מ-Google Drive פגיעות (URLs עם תפוגה, חסמי גישה). יש להשלים מיגרציה מלאה.

2. **טוקן פייסבוק מתחדש אוטומטית**
   הטוקן פג תוקף מדי פעם ומצריך פעולה ידנית. פתרון: שמירת Long-Lived Token + workflow שמחדש אותו.

3. **ניוזלטר — היסטוריית שליחות**
   אין כרגע היסטוריה של מיילים ששלחנו.

### עדיפות בינונית

4. **Pinterest — כיתובים עם Claude**
   Instagram ו-Facebook משתמשים ב-Claude Vision. Pinterest עדיין לא.

5. **המלצות hardcoded**
   הציטוטים בסקשן "לקוחות ממליצים" כתובים ישירות ב-HTML. כדאי לנהל מהאדמין.

6. **תצוגת סטטוס הזמנה ללקוח**
   לקוח שהזמין הדפסה מקבל רק מייל. אין עמוד "מצב ההזמנה שלי".

### עדיפות נמוכה / עתיד

7. **Analytics לחנות**
   כמה אנשים פתחו lightbox → כמה לחצו "רכוש" → כמה השלימו. כרגע אין funnel tracking.

8. **Google Photos → Drive אוטומטי**
   כרגע עמית מעלה ידנית ל-Drive. אינטגרציה ישירה מ-Google Photos תחסוך צעד.
