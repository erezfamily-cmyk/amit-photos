פרק 01
סקירת האתר
מה amitphotos.com עושה, למי הוא מיועד, ומה הוא מציע

amitphotos.com הוא אתר פורטפוליו + חנות דיגיטלית לצלם אמנותי ישראלי בשם עמית ארז. האתר פועל ב-RTL עברית מלאה ומציע שני מוצרים עיקריים:

📥 קבצים דיגיטליים

קנייה ישירה של תמונה דיגיטלית ב-3 רזולוציות (רשת / הדפסה / מלא), הורדה מיידית אחרי תשלום PayPal.

🖼️ הדפסות פיזיות

הזמנת הדפסה (פוסטר / קנבס / מתכת) שנשלחת ישירות לבית הלקוח מ-Gelato — ללא מגע ידני.

חנות הדפסות (Print Shop)

פתוח מה-Lightbox, פועל ב-3 שלבים ויזואליים:

שלב 1
בחירת סוג: פוסטר / קנבס / מתכת — עם תיאור ואייקון לכל אחד. המחיר נשלף חי מ-Gelato API (/api/print/quote).
שלב 2
בחירת גודל + חיתוך: כלי crop שמסתובב אוטומטית לפי יחס הגודל. בדיקת רזולוציה מינימלית — מונעת הזמנה אם התמונה קטנה מדי. הצגת הערת מכס אם מחיר ≥ $75.
שלב 3
פרטי משלוח + תשלום: שם, מייל, טלפון, כתובת (שורה, עיר, מיקוד). PayPal Payment Standard. אחרי תשלום → /api/print/order-complete → Gelato order.

פרק 05
Backend — Cloudflare Worker
API endpoints, לוגיקת עסקים, אבטחה

כל הלוגיקה של השרת רצה בקובץ worker.js אחד, שנפרס ל-Cloudflare Workers.

Authentication Endpoints
POST/api/loginכניסה בסיסמה. בדיקת brute force (5 ניסיונות → נעילה 15 דק'). מחזיר session token (תוקף 8 שעות).
POST/api/logoutמחיקת session מה-DB.
POST/api/forgot-passwordשולח מייל עם קישור איפוס (תוקף 30 דקות) דרך Resend.
POST/api/reset-passwordמאמת token + מעדכן סיסמה ב-D1. מוחק כל ה-sessions הקיימות.
Photos Endpoints
GET/api/photosמחזיר תמונות published. עם ?admin=1 מחזיר הכל (דורש auth).
POST/api/uploadהעלאה ל-R2 + שמירה ב-D1 + כותרת AI אוטומטית (דורש auth).
PATCH/api/photosעדכון כותרת/קטגוריה/published (דורש auth). אם כותרת גנרית — מפעיל Claude.
DEL/api/photos?id=...מחיקה מ-D1 + מ-R2 (דורש auth).
POST/api/fill-titlesמעבר על כל התמונות ללא כותרת עברית → Claude מייצר כותרות (batch, דורש auth).
POST/api/generate-altמייצר alt text עברי לרשימת תמונות (Claude Vision, דורש auth).
POST/api/repair-r2덮어쓰기 קובץ ב-R2 ללא שינוי D1 — לתיקון שגיאות.
Print Shop Endpoints
GET/api/print/catalogמחזיר את קטלוג ההדפסות (סוגים, גדלים, SKUs של Gelato).
POST/api/print/quoteשולח ל-Gelato → מחשב מחיר (×1.6 ומעגל ל-$5).
POST/api/print/upload-cropמעלה גרסת חיתוך של תמונה ל-R2.
POST/api/print/order-completeאחרי PayPal: מאמת PDT → יוצר הזמנה ב-Gelato → שומר ב-D1 → שולח מייל אישור.
GET/api/print/cancel?token=...ביטול הזמנה (בתוך שעה). מבטל ב-Gelato + מעדכן D1.
POST/api/print/webhookGelato webhook → עדכון סטטוס + מייל שילוח ללקוח כש-shipped.
POST/api/print/refresh-statusשולח שאילתה ל-Gelato ומעדכן סטטוס הזמנה ספציפית (דורש auth).
GET/api/print/ordersמחזיר כל ההזמנות מ-D1 (דורש auth).
Customers & Subscribers Endpoints
POST/api/customersשמירת פנייה מטופס "צור קשר" (ציבורי).
GET/api/customersכל הפניות (דורש auth).
POST/api/subscribersהרשמה לניוזלטר — שמירה ב-D1 + מייל ברוך הבא (ציבורי).
GET/api/subscribersכל הנרשמים (דורש auth).
POST/api/newsletterשליחת ניוזלטר לכל הנרשמים (דורש auth).
GET/api/unsubscribe?id=...הסרה מניוזלטר (ציבורי — קישור מהמייל).
Infrastructure Endpoints
GET/api/verify-paymentאימות PayPal PDT + חזרת URL להורדה לפי גודל שנבחר.
POST/api/trigger-workflowמפעיל GitHub Actions workflow (update-photos.yml וכו') ע"י API.
GET/api/analyticsPage views 30 יום + פירוט מדינות (דורש auth).
GET/photos/{key}הגשת תמונה מ-R2 ישירות.
פרק 06
מסד הנתונים — Cloudflare D1
SQLite שמנוהל ע"י Cloudflare, נגיש מה-Worker בלי latency
טבלהעמודות עיקריותתוכן
photos id, title, category, description, r2_key, url, thumbnail, published, created_at כל התמונות. published=1 מופיע בגלריה. r2_key מצביע על R2.
subscribers id, name, email, notes, created_at נרשמים לניוזלטר — כרגע 2 נרשמים.
customers id, name, email, phone, date, type, status, subject, notes, created_at פניות מ"צור קשר". status: ממתין / טופל / סגור.
print_orders id, prodigi_order_id, photo_id, sku, product_label, sell_price, customer_*, address_*, paypal_tx, status, created_at הזמנות הדפסה. status: in_production / shipped / cancelled.
analytics date, page, views Page views יומיים לפי דף.
analytics_countries date, country, views Page views יומיים לפי מדינה (CF-IPCountry header).
sessions token, created_at, expires_at Session tokens לאדמין. נוקו אוטומטית בכל login.
login_attempts ip, count, last_attempt מונה ניסיונות כניסה כושלים לפי IP.
settings key, value הגדרות מערכת — כולל admin_password אם עודכנה.
reset_tokens token, expires_at Tokens לאיפוס סיסמה (תוקף 30 דקות).
📌 מצב נוכחי של הנתונים D1 מכיל ~12 תמונות (המועברות ל-R2). 471 התמונות הנותרות עדיין ב-data/photos.json עם URLs של Google Drive. מיגרציה מלאה ל-R2 בתהליך.

פרק 09
PWA ו-Service Worker
אתר מותקן כאפליקציה, גישה offline

האתר מוגדר כ-Progressive Web App עם תמיכה בהתקנה ו-offline:

📦 Cache Strategy

Cache-first לקבצים סטטיים (CSS/JS/HTML). Network-first ל-photos.json (תמיד עדכני).

🔄 Cache Management

שם Cache: amit-photos-v2. בהפעלה — מוחק caches ישנים. skipWaiting + clients.claim.

📱 Manifest

שם: "עמית | צילום". Display: standalone (כאפליקציה). RTL + lang=he. אייקון SVG מוטמע.

🌐 Cross-Origin

תמונות Google Drive הן cross-origin — לא נשמרות ב-cache (מחמירות עם שרתי Cloudflare R2 פנימיים).

פרק 12
שיפורים מתוכננים — Roadmap
לפי עדיפות: גבוהה / בינונית / נמוכה
● עדיפות גבוהה
#שיפורלמה חשוב
1 השלמת מיגרציה ל-R2 כרגע 459 מתוך 471 תמונות עדיין ב-Google Drive. URLs של Drive עלולים לפוג, לחסם, או להיות איטיים. R2 מהיר, אמין, ובשליטה מלאה.
2 טוקן פייסבוק Long-Lived + חידוש אוטומטי הטוקן פג תוקף ומחייב פעולה ידנית. Workflow שמחדש Token לפני תפוגה = zero-downtime אוטומציה.
3 היסטוריית ניוזלטר אין תיעוד של מיילים ששלחנו — קשה לדעת מה נשלח, מתי, ולמי. טבלת newsletter_sends ב-D1 תפתור זאת.
● עדיפות בינונית
#שיפורפירוט
4 כיתובים חכמים ל-Pinterest Instagram ופייסבוק משתמשים ב-Claude Vision לכיתובים מותאמים. Pinterest מפרסם עם תיאור גנרי בלבד. הוספת Claude Vision לפינטרסט תשפר engagement.
5 המלצות לקוחות מהאדמין כרגע 3 ציטוטים hardcoded ב-HTML. כשיהיו לקוחות אמיתיים, כדאי לנהל המלצות מהאדמין (CRUD ב-D1).
6 בחירה מרובה בגלריה Shift+Click לבחירת טווח תמונות. שיתוף/הורדה/הוספה לסל של כמה בבת-אחת.
7 עמוד מצב הזמנה ללקוח לקוח שהזמין הדפסה מקבל רק מייל. עמוד /order/{id} שמציג סטטוס חי מ-Gelato = שקיפות ללקוח.
8 Funnel Analytics כמה אנשים פתחו Lightbox → כמה לחצו "רכישה" → כמה השלימו תשלום. כרגע אין tracking על funnel זה.
● עדיפות נמוכה / עתיד
#שיפורפירוט
9 Gelato Branded Inserts פתק תודה ממותג עם לוגו + URL בתוך אריזת ההדפסה. דורש Gelato+ מנוי + כניסה לתוכנית בטא.
10 Google Photos → Drive אוטומטי כרגע עמית מעלה ידנית ל-Drive ואז Script שולף. אינטגרציה ישירה Google Photos → R2 תחסוך שלב.
11 תגובות לסטוריז אינסטגרם auto-reply לתגובות על פוסטים — "תודה! תוכלו לרכוש ב-amitphotos.com" עם Claude.
12 כלי Crop מתקדם הוספת rotation + flip לכלי ה-crop בחנות הדפסות. גרירה חלקה יותר על מובייל.
13 Wishlist / Favorites שמירת תמונות מועדפות ב-localStorage עם אפשרות שיתוף. מגביר engagement ומשפר conversion.
14 A/B Testing מחירים בדיקת מחירים שונים לאוכלוסיות שונות לאופטימיזציה של revenue. דורש analytics משופר.
