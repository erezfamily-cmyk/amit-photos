# Newsletter Redesign — Design Spec
**Date:** 2026-05-19
**Status:** Approved

## Overview

Upgrade the newsletter issue page from a text-heavy format to a visual, conversion-focused magazine. Goal: every issue drives readers to amitphotos.com to buy prints.

## New Sections (added to existing layout)

### 1. Wall Mockup — "כך היא נראית אצלך בבית"
- Hero photo displayed inside a stylized "frame on wall" CSS illustration (dark room, wooden floor strip)
- Below frame: price from (`מ-₪290`), two buttons: "🛒 הזמן הדפסה" → `/photos/:hero_photo_id` and "👁 מידות וחומרים"
- Footer note: "קנבס · אלומיניום · נייר ארכיוני · משלוח עד הבית"
- No new data — uses existing `c.hero.photo_id` and `c.hero.photo_url`

### 2. Gallery Strip — "עוד מהסדרה"
- Horizontally scrollable strip of 3 additional published photos (not the hero)
- Each thumb: photo thumbnail (120×80, object-fit:cover) + Hebrew title below
- Last item: "+N לגלריה" pill linking to `/` (main gallery)
- Server picks 3 photos via `nlPickGalleryPhotos(env, heroPhotoId)`: `SELECT id, title, r2_key FROM photos WHERE published=1 AND id != ? ORDER BY RANDOM() LIMIT 3`
- Photo URL: `https://amitphotos.com/photos/:r2_key` (existing pattern)
- Stored in `content_json` as `gallery_photos: [{id, title, url}]` — picked at generation time

### 3. Sale Banner — "מבצע החודש"
- Banner with: discount tag, title, description, crossed-out price, sale price, CTA button
- Stored in `content_json` as `sale: {title_he, desc_he, original_price, sale_price, discount_label}`
- Generated by Claude (see prompt additions below)
- CTA button links to `/` (main site) — Amit can edit in admin editor
- Only shown in `full` issues (not flash)

### 4. Step-by-Step Guide
- Replace single `guide.text_he` paragraph with 3 numbered steps
- Each step: `{num, title_he, text_he}` — no photo per step
- `content_json.guide.steps: [{num:1, title_he, text_he}, ...]`
- Navigation: 3 pill buttons (שלב 1/2/3) — clicking shows that step's content (JS toggle)
- Active step pill: gold background. Inactive: dark.
- Fallback: if `guide.steps` missing (old issues), show `guide.text_he` as before

### 5. Home / Office CTA Cards
- 2×2 grid, static content, links to site sections:
  - 🛋️ לסלון/חדר שינה → `/?category=נוף`
  - 💼 למשרד/קליניקה → `/?category=עירוני`
  - 🎁 מתנה מיוחדת → `/contact/`
  - ✨ הדפסה אישית → `/contact/`
- Shown in all issue types (full + flash)

### 6. Personal Contact Card — "רוצה עזרה אישית?"
- Card with Amit's avatar emoji (👨‍🎨), intro text, three action buttons:
  - 💬 **וואטסאפ ישיר** → `https://wa.me/972503333227`
  - 💳 **ביט** → `https://bit.ly/amit-photos` (placeholder — Amit to provide)
  - 💳 **פייבוקס** → `https://payboxapp.page.link/amit` (placeholder — Amit to provide)
- Footer note: "תשלום נוח — ביט, פייבוקס, או כרטיס אשראי באתר"
- Static — same on every issue. WhatsApp: `+972503333227`
- Shown in all issue types

## Claude Prompt Additions

In `nlGenerateContent`, add to the JSON schema:

```
"sale": {
  "title_he": "שם המבצע (מקסימום 5 מילים)",
  "desc_he": "תיאור קצר (עד 10 מילים, כולל תאריך סיום)",
  "original_price": "₪480",
  "sale_price": "₪385",
  "discount_label": "20% הנחה"
},
"guide": {
  "title_he": "...",
  "title_en": "...",
  "slug": "...",
  "steps": [
    {"num": 1, "title_he": "שם השלב", "text_he": "הסבר קצר (2-3 משפטים)"},
    {"num": 2, "title_he": "...", "text_he": "..."},
    {"num": 3, "title_he": "...", "text_he": "..."}
  ]
}
```

Sale is creative — Claude invents a realistic promo (e.g., "מבצע קיץ על הדפסות A2"). Amit can edit in the admin editor.

## Backward Compatibility

- Gallery strip: shown only if `c.gallery_photos?.length`
- Sale banner: shown only if `c.sale`
- Guide: falls back to `guide.text_he` if `guide.steps` missing
- No data migration needed — old issues display as before, new issues get full layout
