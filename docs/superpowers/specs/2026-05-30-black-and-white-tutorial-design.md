## מטרה

ללמד את החשיבה של צלם שחור-לבן: מה הופך תמונה לעוצמתית כשמסירים את הצבע, ואיך לזהות את זה עוד בשטח — לפני הצילום.

---

## ארכיטקטורה

קובץ יחיד: `camera/black-and-white/index.html`  
אין תלויות חיצוניות מעבר למה שקיים בפרויקט (Heebo + Syne, CSS variables, nav.js).  
תמונות נטענות מ-Google Drive/Photos כמו שאר הגלריה.  
שחור-לבן מושג ב-CSS בלבד: `filter: grayscale(100%)`.

## מבנה הדף

### 1. Hero
- כותרת: "צילום שחור-לבן — מה הופך תמונה לעוצמתית?"
- תגית: "מדריך למתחיל"
- תקציר: "כשמסירים את הצבע, מה נשאר? אור, צורה, מרקם, סיפור."
- דו-לשוני: `data-he` / `data-en` + `setLang()`

### 2. "למה שחור-לבן?" (סעיף טקסט)
הסבר קצר ונקי:
- הסרת הצבע מאלצת את הצופה להתמקד בצורה, אור, מרקם
- תמונה חלשה — שחור-לבן מגדיל את החולשה
- תמונה חזקה — שחור-לבן מגדיל את העוצמה

### 3. דמו אינטראקטיבי — "בוחן הניגוד" (3 כרטיסיות)

**מנגנון:** Drag-to-reveal slider — קו שגוררים ימין/שמאל ישירות על התמונה.  
שמאל = צבע, ימין = שחור-לבן (CSS grayscale).  
מתחת לכל תמונה: שורת לקח אחת, פשוטה.

**3 תמונות מהגלריה של עמית:**

| # | תמונה | ID | לקח |
|---|---|---|---|
| 1 | שדה הגלגלים הצהוב | `1LDJDivmCJtNCpb1NhGpV5OQeyfjeh1f2` | צהוב+ירוק = אותו אפור. ניגוד הצבע נעלם בשחור-לבן |
| 2 | מגדל דוד בירושלים | `1ySQLVxa-NZVG-WGwcTi6eMp_aHR80vo8` | צורה + צל = ניגוד חזק. עובד מצוין בשחור-לבן |
| 3 | דיוקן בצבעים חיים | `1iGsR7oUKjZ75jOxcqdw-plYU72f1LbEp` | עור + אור = מרקם. פורטרט מנצח בשחור-לבן |

**מימוש slider (ללא dependencies):**
- Container עם `position: relative; overflow: hidden`
- תמונת צבע כ-background, תמונת שחור-לבן (אותה תמונה + `filter: grayscale(1)`) כ-overlay עם clip-path
- `mousedown/mousemove/touchmove` זזים את קו ה-divider
- קו divider: `::after` pseudo-element לבן עם `cursor: ew-resize`
- Mobile friendly: touch events

### 4. "4 דברים לחפש לפני הצילום"
רשימה עם אייקון לכל פריט:
- 🌑 **ניגוד טון** — בדוק: האם האובייקט כהה מהרקע? אם הצבעים דומים בבהירות, הם ייראו אותו גוון
- ✨ **אור ממוקד** — צל חד, אור שמגיע מכיוון אחד = ניגוד טבעי
- 🧱 **מרקם** — עץ, אבן, עור, בד — מרקמים מתחדדים בשחור-לבן
- 📐 **צורה וקומפוזיציה** — קווים, עקומות, גיאומטריה — כשהצבע לא מסיח, הצורה בולטת

### 5. גלריה קטנה — תמונות שחור-לבן של עמית
הצגת 4-6 תמונות מקטגוריית "שחור-לבן" בגלריה פשוטה (grid),  
עם לינק לגלריה המלאה.

### 6. ציוד מומלץ (Affiliate)
כמו שאר המדריכים — 2-3 כרטיסי gear:
- פילטר ND (לחשיפות ארוכות בשחור-לבן)
- עדשת 50mm/85mm (פורטרט שחור-לבן)
- Skylum Luminar / Lightroom (עריכת שחור-לבן)

## רכיבי UX

- **Slider handle:** עיגול לבן עם חץ דו-כיווני ←→ + hint טקסט "גרור" שנעלם אחרי גרירה ראשונה
- **Mobile:** touch events, slider רחב יותר לנגיעה קלה
- **Loading:** תמונות נטענות lazy, placeholder אפור עד לטעינה
- **A11y:** `aria-label` על הslider, alt text על התמונות
